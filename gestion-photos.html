Instructions pour Ajouter les Photos

  1. Organisez vos photos dans les dossiers suivants selon la catégorie du produit
  2. Nommez chaque photo avec le nom exact du produit (ex: tapo.png, z24.png, etc.)
  3. Utilisez des formats d'image supportés : .png, .jpg, .jpeg
  4. Recommandé : Taille optimale 800x800px ou plus pour une meilleure qualité
  5. Une fois la photo ajoutée dans le bon dossier, elle apparaîtra automatiquement sur le site

Ajouter un Nouveau Produit

Pour ajouter un nouveau produit :

  1. Ajoutez la photo dans le dossier approprié avec le nom du produit
  2. Modifiez le fichier produits.html pour ajouter la carte produit
  3. Ajoutez les détails dans js/product-detail.js
  4. Mettez à jour cette page avec le nouveau produit
